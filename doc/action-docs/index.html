Description

GitHub Action for creating a repository release using RTK.

Inputs

| name | description | required | default | | --- | --- | --- | --- | | release_type |

RTK release type, options are 'major', 'minor', 'patch'

| false | minor | | github_token |

GitHub token

| true | "" | | git_user_name |

Git user name

| false | pakkunbot | | git_user_email |

Git user email

| false | pakkunbot@users.noreply.github.com | | push_changes |

Whether to push release commit and tags to remote repository

| false | true |

Runs

This action is a composite action.

Usage

```yaml - uses: PROJECT@VERSION with: release_type: # RTK release type, options are 'major', 'minor', 'patch' # # Required: false # Default: minor

github_token:
# GitHub token
#
# Required: true
# Default: ""

git_user_name:
# Git user name
#
# Required: false
# Default: pakkunbot

git_user_email:
# Git user email
#
# Required: false
# Default: pakkunbot@users.noreply.github.com

push_changes:
# Whether to push release commit and tags to remote repository
#
# Required: false
# Default: true

```
